## 自行推送使用方式
- 请自行安装python 3，如果已安装版本是python 2，推荐使用版本管理工具。
```buildoutcfg
python3：https://www.python.org/downloads/
```
- 安装所需依赖
```buildoutcfg
pip install wxpy
pip install webdriver
pip install pillow
如果还有其他缺失 请根据提示 pip install 安装
```
- 运行src下的watcher.py
- 运行wechat_push.py
- 扫码登陆
- 开始推送
